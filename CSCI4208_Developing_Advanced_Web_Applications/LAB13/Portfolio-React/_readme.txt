How to use HTTP server to host HTML files to Browser

Option 1 (Recommended)
Chrome Extension:
Tutorial:     https://www.youtube.com/watch?v=AK6swHiPtew
Chrome App:   https://chrome.google.com/webstore/detail/web-server-for-chrome/ofhbbkphhbklhfoeikjpcbhemlocgigb?hl=en



Option 2:
Python2:
Step 1: Open terminal at folder containing the index.html
Step 2: execute command: python -m SimpleHTTPServer



Option 3:
Python3:
Step 1: Open terminal at folder containing the index.html
Step 2: execute command: python3 -m http.server



Option 4:
PHP:
Step 1: Open terminal at folder containing the index.html
Step 2: execute command: php -S localhost:8000
